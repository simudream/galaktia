# Introduction #

We use [nose](http://somethingaboutorange.com/mrl/projects/nose/0.11.1/) as testing framework.

# Testing #

## Command cheatsheet ##

  * To install nose:
> ` easy_install nose `

  * To run tests:
> ` nosetests -vs path_to_tests `

  * Command usage help:
> ` nosetests -h `
